Git & GitHub Basics

What is Git?

Git is a distributed version control system that allows multiple developers to work on a project simultaneously without overwriting each other's changes. It was created by Linus Torvalds in 2005.

Key features of Git include:

  • Branching and merging
  • Distributed development
  • Version tracking
  • Staging area

What is GitHub?

GitHub is a platform for version control and collaborative software development. It builds upon Git by providing a web-based interface and additional features.

GitHub features include:

  • Remote repositories
  • Pull requests
  • Issue tracking
  • Code review tools
